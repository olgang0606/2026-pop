# 2026-pop
# 2026년 대한민국 인구·다문화 대시보드

2026년 6월 기준 **연령별 인구 구조**와 **다문화(외국인) 현황**을 한눈에 볼 수 있는 Streamlit 웹 애플리케이션입니다.

## 🚀 주요 기능

### 1. 연령별 한국인 인구 구조
- 전국 시도·시군구·읍면동 단위까지 상세 데이터 제공
- 선택 지역의 **연령별 인구 피라미드(꺾은선 그래프)**
- 성별(남/여/총) 비교

### 2. 다문화(외국인) 현황
- **등록 외국인 현황** (지역별, 체류자격별, 국적별)
- **신규 유입·유출 통계** (2026년 6월)
- 순증감 분석

### 3. 🧬 쌍둥이동네 찾기 (인구 구조 유사도 분석)
- 선택한 지역과 **전국적으로 인구 구조가 가장 비슷한 지역 TOP5** 자동 추천
- 코사인 유사도 기반
- 비교 그래프 제공

## 🛠 기술 스택

- **Frontend**: Streamlit
- **Visualization**: Plotly
- **Data Processing**: Pandas, NumPy
- **Similarity**: Cosine Similarity (NumPy 구현)
